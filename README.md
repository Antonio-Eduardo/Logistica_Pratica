 Sistema de Logística em Java

Projeto desenvolvido para simular o gerenciamento de encomendas com controle de status.

Funcionalidades
- Cadastro de encomendas
- Diferenciação entre encomendas nacionais e internacionais
- Cálculo de taxas e impostos 
- Atualização de status da encomenda 
- Aplicação de regras de negócio com services
- Busca de encomenda pelo codigo de rastreio
- Atualização do status da encomenda e historico com todos os status daquela encomenda

Conceitos aplicados
- Programação Orientada a Objetos (POO)
- Herança e polimorfismo
- Enum para controle de status
- Separação de responsabilidades (Services)
- Tratamento de exceções

 Tecnologias
- Java
- Git

 Como executar
- Rodar a classe principal (Programa.java)
